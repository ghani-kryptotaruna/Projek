Buat 3 Virtualmechine dengan ip sebagai berikut:
VM1 = load balancer 192.168.4.250
VM2 = APP1 192.168.4.251
VM3 = APP1 192.168.4.252

lalukan update di semua VM dengan perintah:
$ sudo apt update && sudo apt upgrade -y

Atur waktunya menjadi Asia/Jakarta dengan perintah:
$ sudo timedatectl set-timezone Asia/Jakarta

Lalu cek dengan perintah
$ date
Sat Mar 28 02:19:53 PM WIB 2026

Ubah masing masing hostname

Masuk ke VM1 lalu install dan konfigurasi haproxy
$ sudo apt update
$ sudo apt install haproxy keepalived -y

$ sudo vim /etc/haproxy/haproxy.cfg
global
    log /dev/log local0
    maxconn 2048

defaults
    mode http
    timeout connect 10s
    timeout client  300s
    timeout server  300s

frontend http_front
    bind *:80
    default_backend nextcloud_cluster

backend nextcloud_cluster
    balance roundrobin
    option httpchk

    default-server inter 2s fall 2 rise 1

    server vm2 192.168.4.251:80 check
    server vm3 192.168.4.252:80 check

$ sudo systemctl restart haproxy

$ sudo vim /etc/keepalived/keepalived.conf
global_defs {
   script_security 1
}

vrrp_instance VI_1 {
    state MASTER
    interface enp0s3
    virtual_router_id 51
    priority 100
    advert_int 1

    authentication {
        auth_type PASS
        auth_pass ncpass   # max 8 karakter!
    }

    virtual_ipaddress {
        192.168.4.100
    }
}                                    20,1          All

$ sudo systemctl enable keepalived
$ sudo systemctl start keepalived
